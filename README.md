## Run Locally

Clone the project

```bash
  git clone https://github.com/JoelDeonDsouza/Lena.git
```

Go to the project directory

```bash
  cd lena
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
